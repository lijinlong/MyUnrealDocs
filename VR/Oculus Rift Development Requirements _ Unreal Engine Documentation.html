Recommended Desktop Hardware

See Oculus' System and Hardware Requirements
Hardware and Software Specifications

Supported VR Headsets

Software Requirements

Desktop Operating System

Windows 7 or later
MacOS 10.10 or later (x86 only)

SDK Version

Oculus Runtime 1.28

The currently supported SDK version(s) can be found in UEBuildTODO.cs. You can find it in your [UE4 Root Director]/Engine/Source/Programs/UnrealBuildTool/Platforms/TODO folder.
